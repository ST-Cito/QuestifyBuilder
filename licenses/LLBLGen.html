Pricing details

The price of LLBLGen Pro is €299.- per seat, exclusive of VAT and handling costs. This gets you a perpetual license and a 1-year subscription.
The license is valid forever: it won't expire if your subscription expires. If your subscription is expired, it can be used with all v5.x or higher versions that were available at the expiration date.
A subscription gives you access to fix versions, hotfixes, new major / minor versions and alpha builds and is valid for one year. The renewal cost is €199.- per year, and a subscription can be renewed at any time, whether it's expired or not. You'll get the first year of subscription for free when purchasing licenses.
For an extra €99.- ORMProfiler is added to your data-access toolbox.

A valid license is also required if you use the command line tools CliGenerator and the CliSyncWithDatabase. This means that if you're using these tools in a continuous integration system or build system, you have to purchase an additional license for that system.

Volume discounts are available if you purchase multiple licenses at once. The correct total amount for the number of licenses you purchase is calculated automatically on the order page.

If you would like to purchase a site license instead of per seat licenses (interesting for teams of over 50 people), please contact us at sales@llblgen.com for more information.

How do I upgrade?

If you have 4.x licenses, you can upgrade to v5.x for €199.- per developer seat, after logging in by choosing 'Upgrade to latest version' from the 'My Account' menu.

How do I renew my subscription?

You can renew your subscription for €199.- per developer seat, after logging in, by choosing 'Renew your subscriptions' from the 'My Account' menu.

Are there any hidden costs?

There are no additional or hidden costs. If your subscription has expired, you'll still be able to download the software released before the expiration date (but not newer software, including fixes).
Customer support is free.

When can I expect my licenses?

After your payment is complete, you'll receive two seperate e-mails: one with the link to the URL where to download your copy of LLBLGen Pro and another one with your licence file(s) and the login information for the Customer Area on our site.
If you've paid using a credit card, you should receive these emails within minutes. If you don't, please contact us at sales@llblgen.com.

Resellers

If you are a reseller and wish to purchase LLBLGen Pro for a customer, please send an e-mail to info@llblgen.com for more information.

Access to source code

Customers have access to the LLBLGen Pro Runtime Framework source code, all task performers sourcecode (e.g. the code generator engines), the shipped plugins and the drivers sourcecode. The runtime sourcecode comes with a license which allows you to distribute a modified version of the LLBLGen Pro Runtime Framework in binary form. Customers also get access to the source code of several SDK tools.

EU customers

If you are in the Netherlands or another EU country and prefer paying through bank transfer, please send an email to sales@llblgen.com for the details and an invoice.
